Happy Number(測驗二題目)

題目

判斷一個正整數N是否為Happy Number。Happy number的定義: 輸入的數字稱作S0，S1是將S0的每個位數的數字作平方和，S2則是S1的每位數字的平方和。比照辦理，若Si能收斂成1，則S0就稱為快樂數字(Happy number)。若數列S無法收歛成1，只能在迴圈裡反覆變化者，則S0稱為不快樂數字(Unhappy number)。
例如 :

7 -> 49 -> 97 -> 130 -> 10 -> 1，則7為Happy number。

(解釋說明: 49為72，97為42+92，130為92+72，10為12+32+02，1為12+02)

4 -> 16 -> 37 -> 58 -> 89 -> 145 -> 42 -> 20 -> 4，則4為Unhappy number。

Input
每組測試資料一列含有 1 個正整數 N（ N < 109） Output 輸出 N 為 Happy number 或 Unhappy number

7 => 7 is a Happy number 4 => 4 is an Unhappy number 13 => 13 is a Happy number

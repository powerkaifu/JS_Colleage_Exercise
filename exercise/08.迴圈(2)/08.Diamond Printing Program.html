Diamond Printing Program

題目

Write a program to read an odd number in the range of 1 to 19 to specify the number of rows in the diamond, and print the corresponding diamond shape.
For example, the input number is 5, the corresponding diamond is shown as below:

  ★
 ★★★
★★★★★
 ★★★
  ★

You may use printf statements that print either a single asterisk (★) or a single blank.
Maximize your use of repetition (with nested for statements) and minimize the number of printf statements.
